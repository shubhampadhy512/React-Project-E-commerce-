## Description
YourMart is a clone of an e-commerce website built using React.
It fetches product data from the Dummy API (dummy-item) and manages state using Redux Toolkit.
This project demonstrates a shopping cart, product listing, and interactive UI.

## Features
- Browse products fetched from Dummy API
- Add products to the shopping cart
- Manage product quantities using Redux Toolkit
- Responsive design built with React
- Clone of a typical e-commerce website

## Tech Stack
- **Frontend:** React
- **State Management:** Redux Toolkit
- **API:** Dummy API (dummy-item)
- **Other:** CSS, JavaScript
